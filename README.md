このプロジェクトについて  
---
学校の百葉箱をラズパイを用いて自動化を行い、  
温度湿度及び気圧の測定・データの蓄積、利用をするものです。  

環境の説明
---  
基本プログラムはPythonで書くほか、データの利用にAPI等を利用する予定です。  
Python 3.7  
試験用1,2号機...Raspberry Pi Model B  
試験用3,4号機...Raspberry Pi Zero HW  
百葉箱箱は2機運用の予定です。
